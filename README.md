# award-be

### Run DB script from
```
    ./db_init.sql
```

### Rename env.example and change DB env
```
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=award_db
    DB_USERNAME=root
    DB_PASSWORD=
```

## Project setup
```
composer install
```

### Compiles and hot-reloads for development
```
php artisan serve --port=8000
```
